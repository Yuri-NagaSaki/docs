## What is ImageFlow?

ImageFlow is a modern, full-stack image service system designed for high-performance image management and distribution. Built with Go and Next.js, it automatically optimizes images for different devices and browsers while providing a beautiful, intuitive interface for image management.

## Architecture Overview

ImageFlow follows a modern microservices architecture:

### Backend (Go 1.23+)
- **High-Performance Processing**: Uses libvips for lightning-fast image conversion
- **Async Worker Pool**: Background processing with configurable concurrency
- **Redis Metadata**: Fast metadata storage with file-based fallback
- **API Authentication**: Secure API key-based authentication system

### Frontend (Next.js 14)
- **Static Export**: Pre-built and embedded in the Go binary for single-file deployment
- **Responsive Design**: Mobile-first design with device detection
- **Real-time Updates**: Live upload progress and instant image management

### Storage Options
- **Local Storage**: Perfect for development and small deployments
- **S3 Compatible**: Production-ready with any S3-compatible storage service
- **Custom Domains**: Support for CDN and custom domain configurations
