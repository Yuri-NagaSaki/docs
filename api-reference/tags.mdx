## Get All Tags

<api-endpoint method="GET" endpoint="/api/tags">

Retrieve a list of all available tags in your image collection.

### Authentication Required
<param-field header="X-API-Key" type="string" required>
  Your ImageFlow API key
</param-field>

### Response

```json
{
  "success": true,
  "data": {
    "tags": [
      {
        "name": "nature",
        "count": 45,
        "created_at": "2023-10-15T10:30:00Z"
      },
      {
        "name": "landscape", 
        "count": 32,
        "created_at": "2023-10-16T14:20:00Z"
      }
    ],
    "total": 2
  }
}
```

### Examples

```bash
curl -H "X-API-Key: your-key" https://your-domain.com/api/tags
```

</api-endpoint>
