## Trigger Cleanup

<api-endpoint method="POST" endpoint="/api/trigger-cleanup">

Manually trigger cleanup of expired images and orphaned files.

### Authentication Required
<param-field header="X-API-Key" type="string" required>
  Your ImageFlow API key
</param-field>

### Response

```json
{
  "success": true,
  "data": {
    "cleanup_started": true,
    "job_id": "cleanup_abc123",
    "estimated_duration": "5-10 minutes"
  },
  "message": "Cleanup process started successfully"
}
```

### Examples

```bash
curl -X POST \
  -H "X-API-Key: your-key" \
  https://your-domain.com/api/trigger-cleanup
```

</api-endpoint>
