## List Images

<api-endpoint method="GET" endpoint="/api/images">

Retrieve a list of uploaded images with filtering and pagination options.

### Authentication Required
<param-field header="X-API-Key" type="string" required>
  Your ImageFlow API key
</param-field>

### Query Parameters

<param-field query="tag" type="string" optional>
  Filter by tag
</param-field>

<param-field query="limit" type="integer" optional>
  Number of images to return (default: 50, max: 200)
</param-field>

<param-field query="offset" type="integer" optional>
  Number of images to skip (for pagination)
</param-field>

### Examples

```bash
curl -H "X-API-Key: your-key" "https://your-domain.com/api/images?limit=10"
```

</api-endpoint>
