## Get Configuration

<api-endpoint method="GET" endpoint="/api/config">

Retrieve current system configuration and settings.

### Authentication Required
<param-field header="X-API-Key" type="string" required>
  Your ImageFlow API key
</param-field>

### Response

```json
{
  "success": true,
  "data": {
    "upload_limits": {
      "max_file_size": 10485760,
      "max_files_per_request": 20,
      "max_total_size": 104857600
    },
    "supported_formats": ["jpeg", "png", "webp", "avif", "gif", "tiff"],
    "image_quality": 80,
    "worker_threads": 4
  }
}
```

### Examples

```bash
curl -H "X-API-Key: your-key" https://your-domain.com/api/config
```
