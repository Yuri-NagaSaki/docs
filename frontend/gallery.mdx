## Image Gallery

The ImageFlow gallery provides a beautiful, responsive interface for browsing and managing your image collection. Built with performance in mind, it handles large collections efficiently.

## Features

<CardGroup cols={2}>
  <Card title="Masonry Layout" icon="grid-3x3">
    Responsive masonry grid that adapts to different screen sizes
  </Card>
  
  <Card title="Advanced Filtering" icon="filter">
    Filter by tags, date, format, and other metadata
  </Card>
  
  <Card title="Lazy Loading" icon="zap">
    Optimized performance with progressive image loading
  </Card>
  
  <Card title="Full-screen Viewer" icon="maximize">
    Modal viewer with keyboard navigation and zoom
  </Card>
</CardGroup>

## Gallery Components

- **ImageCard**: Individual image display with hover effects
- **ImageModal**: Full-screen image viewer
- **ImageFilters**: Advanced filtering interface
- **MasonryLayout**: Responsive grid system
- **LoadingSpinner**: Progressive loading states
