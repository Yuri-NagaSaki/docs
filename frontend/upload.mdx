## Upload Interface

ImageFlow provides an intuitive drag-and-drop upload interface that makes uploading multiple images effortless. The interface supports batch uploads, real-time progress tracking, and immediate preview of selected files.

  <Card title="Drag & Drop" icon="hand">

  <Card title="Batch Upload" icon="layers">

  </Card>

## Upload Components

The upload system consists of several interconnected components:

- **UploadDropzone**: Main drop area for file selection
- **ImagePreviewGrid**: Shows selected images before upload
- **UploadProgress**: Real-time progress tracking
- **TagSelector**: Interface for adding tags
- **ExpirySelector**: Set expiration dates
